Note:
Ask for generic info. during sign-up
Then when going for a specific action like recommending schemes and eligibility checking, give fields to fill all the data, but autofill whatever data is already provided in the profile



## Personal Details
 - Full Name
 - Date of Birth / Age
 - PAN Number
 - Mobile Number
 - Email Address
 - Current Address with PIN code

## Employment Information
 - Employment Type (Salaried/Self-employed/Business)
 - Employer/Business Name
 - Years with Current Employer/In Business
 - Monthly Gross Income
 - Monthly Net Income (Take-home)

## Financial Information
 - Existing Monthly EMI (if any)
 - Total Outstanding Loan Amount
 - Number of Active Loans
 - Monthly Fixed Expenses
 - Bank Account Number
 - IFSC Code
 - Any Past Loan Defaults (Yes/No)

## Additional Fields
 - Marital Status
 - Number of Dependents
 - Years at Current Address

## Loan Requirements (To be submitted while searching for a scheme, not in the user profile)
 - Loan Type (Personal/Home/Education/Business)
 - Desired Loan Amount
 - Preferred Tenure (in months)
 - Purpose of Loan
 - Credit Information
 - Credit Score (Self-reported or calculated by the simulator)
