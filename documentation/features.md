# Platform Features
## Core Engine
  - *Smart Multi-Bank Loan Recommendation Engine*  
    Ranks eligible loan schemes using defined financial factors.
    
## Input & Interaction
  - *Unified Form + Chatbot Interface*  
    Structured form captures essential data, chatbot gathers preferences.  
  - *Single Unified Submission*  
    Both inputs submitted together for streamlined processing.
    
## Loan Evaluation
  - *Eligibility Filtering*  
    Rule-based checks per bank.  
  - *EMI Calculator*  
    Monthly repayment estimation.  
  - *Total Repayment Calculator*  
    Calculates principal + interest over tenure.
    
## Comparison & Benchmarking
  - *Loan Scheme Comparison*  
    Side-by-side view of eligible schemes.  
  - *Interest Rate Benchmarking*  
    Visual comparisons inspired by CPU benchmark sites.  
  - *Documentation Requirement Listing*  
    Scheme-specific document checklist.

## Decision Factors (Core 5)
1. Documentation Burden  
2. Principal Reduction Flexibility  
3. Early Loan Closure Impact  
4. EMI Affordability  
5. Total Repayment Cost  

## Insights & Explainability
  - *Loan Decision Insights*  
    Balances cost, comfort, flexibility, and long-term impact.  
  - *Early Closure Impact Analysis*  
    Highlights savings or penalties on early repayment.
    
## Transparency Assistant
  - *RAG-Based Scheme Chatbot*  
    Answers scheme-related queries in real time.  
  - *Hidden Charges Detection*  
    Flags processing fees, foreclosure clauses, and penalties.
